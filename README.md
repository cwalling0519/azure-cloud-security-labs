# Azure Cloud Security Engineering Portfolio – Corey Walling

This repository contains hands-on Azure security labs focused on identity protection, network segmentation, governance, monitoring, and least privilege design.

These labs were built while preparing for the AZ-104 certification and developing practical cloud security engineering skills.

---

## Security Skills Demonstrated

- Microsoft Entra ID hardening
- Conditional Access & MFA enforcement
- Role-Based Access Control (RBAC)
- Network segmentation (VNets, NSGs)
- Private endpoints & reduced public exposure
- Azure Policy enforcement
- Log Analytics & alerting
- Cost governance controls

---

## Labs

### Lab 01 – Secure Network Segmentation
Design and deployment of segmented Azure infrastructure with NSGs, Bastion access, and least privilege RBAC.

### Lab 02 – Identity & Zero Trust Implementation
Configuration of Entra ID users, RBAC, Conditional Access policies, and governance controls.

### Lab 03 – Monitoring & Detection Engineering
Implementation of Azure Monitor, Log Analytics, alert rules, and budget governance.
